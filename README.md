# soal001
Scanner digunakan untuk membaca input dari keyboard.


Program meminta pengguna memasukkan sebuah nilai (masukkan.nextInt();).


if (nilai % 2 == 0) artinya: jika sisa pembagian nilai dengan 2 adalah 0 → bilangan genap.


Jika tidak (else), maka bilangan ganjil.


Program menampilkan hasil sesuai kondisi tersebut.


 Contoh:


Input: 8 → Output: bilangan genap


Input: 7 → Output: bilangan ganjil
